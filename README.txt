Django Version: 2.11

Python Version: 3.7.0

Steps to create a project template:

  * In CMD:

  1. django-admin startproject website;

  2. cd website

  3. py manage.py startapp music

  4. create all files by hand based on my log and copy-paste(including website/setting.py, website/urls.py)

  5. py manage.py makemigrations

  6. py manage.py migrate

  7. py manage.py runserver
